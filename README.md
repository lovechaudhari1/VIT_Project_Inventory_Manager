# Simple Inventory Management System

A straightforward, console-based inventory program written in Python.  
Perfect for beginners learning classes and dictionaries, or for anyone who needs a lightweight tool to track items in a small store, warehouse, home pantry, or personal collection.

## Features

- Add new items with a unique code, name, quantity, and price  
- Update stock levels — add more units or remove them (use negative numbers to subtract)  
- Prevent accidental over-removal (you can’t go below zero stock)  
- View detailed information for a single item by its code  
- List the entire inventory in a clean, readable table  
- Input validation and helpful error messages  
- Simple, clear menu that’s easy to navigate  

## Requirements

- Python 3.6 or newer (no extra libraries needed)

## How to Run

1. Copy the code into a file named inventory.py (or any name you prefer).
